# v0.1.18 - Commercial S01 Domain/App Canonical Commercial Catalog

- Date: 2026-04-10
- Version: v0.1.18
- Type: patch
- Scope: commercialization `S01` canonical catalog convergence

## Delivered

- added shared commercial catalog semantics in `sdkwork-api-domain-catalog`:
  - `ApiProduct`
  - `ProductOffer`
  - `CatalogPublication`
  - `CommercialCatalog`
  - explicit `ApiProductKind / QuoteKind / CatalogPublicationKind / CatalogPublicationStatus`
- added `sdkwork-api-app-catalog` canonical seed builder and deterministic product/offer/publication IDs
- rewired `sdkwork-api-app-commerce` so portal catalog `products / offers` now come from the canonical domain/app model instead of local assembly
- preserved outward portal catalog compatibility fields and avoided storage-schema churn
- repaired sqlite compile blockers needed for fresh-loop verification without widening the feature scope
- removed the obsolete provider-account tuple decoder path left behind by the sqlite-row cutover
