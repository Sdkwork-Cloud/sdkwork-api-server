# v0.1.25 - Commercial S01 Publication-Revision Governance

- Date: 2026-04-10
- Version: v0.1.25
- Type: patch
- Scope: commercialization `S01` publication revision governance convergence

## Delivered

- upgraded canonical `CatalogPublication` with additive `publication_revision_id / publication_version / publication_source_kind / publication_effective_from_ms`
- added deterministic commercial publication revision identity so publication governance no longer hides inside `pricing_plan_version`
- threaded publication revision evidence through portal catalog offers, commerce quotes, order snapshots, order read views, and portal OpenAPI while preserving existing `pricing_plan_*` compatibility
