## Delivered Changes

### Verification Recovery

- added `sdkwork-api-ext-provider-native-mock` as an explicit admin test dependency so the native dynamic fixture library is built and discoverable from the admin test binary
- switched admin test support to import `FIXTURE_EXTENSION_ID` from the native mock crate instead of duplicating the string locally
- serialized the extension package discovery tests that mutate shared `SDKWORK_EXTENSION_*` environment variables, removing the race that caused intermittent zero-package assertions

### Repository Hygiene

- extended `.gitignore` to cover:
  - `/target-*/`
  - `/tmp/`
  - `/bin/.sdkwork-target/`
- this keeps local verification targets and temporary script artifacts out of Git status while preserving all real source, docs, and release assets for commit
