# v0.1.38 - Commercial S06 Public Market/Coupon/Commercial API Convergence

- Date: 2026-04-10
- Version: v0.1.38
- Type: patch
- Scope: commercialization `S06` public gateway market/coupon/commercial convergence

## Delivered

- added authenticated public gateway routes for:
  - market catalog and quote
  - coupon validate / reserve / confirm / rollback
  - commercial account and benefit-lot visibility
- exposed outward coupon semantics through `template / campaign / applicability / effect`, keeping public coupon effect explicit as `checkout_discount | account_entitlement`
- exposed `scope_order_id` on public commercial benefit-lot payloads so account-arrival evidence can be correlated to coupon redemptions
- published the same 9-route surface into gateway `/openapi.json` under the new tags `market`, `marketing`, and `commercial`

## Known Limits

- `GET /commercial/account/benefit-lots` still scans store-wide lots before account filtering; account-scoped query/pagination remains the next scale fix
- downstream SDK/client regeneration from the updated public gateway OpenAPI has not been replayed in this slice
